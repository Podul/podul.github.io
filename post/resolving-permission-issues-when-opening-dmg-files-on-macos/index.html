解决 macOS 上无法打开 DMG 文件的权限问题

2024-01-25 · Podul

问题描述

当尝试打开 .dmg 文件时，遭遇错误提示：“未能完成该操作。权限被拒绝”。

系统环境

  • 设备类型：Mac mini
  • 系统安装方式：通过外置硬盘安装操作系统
  • 操作系统版本：macOS Sonoma 14.0

解决方法

首先在终端中执行以下命令：sudo xattr -r -d com.apple.quarantine /path/to/dmg。这将移除文件的隔离属性。完成此操作后，再次尝试打开 .dmg 文件即可。
